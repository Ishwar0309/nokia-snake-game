# 🐍 Nokia Snake Game
This classic game is reimagined and coded in Python using a double-ended queue (deque).

# 📦 Installation & Usage
```
git clone https://github.com/Ishwar0309/nokia-snake-game.git
cd nokia-snake-game
python3 snakes.py
```
# 🚀 Usage
Use the follwing keys to control the snake:
```
W: Move Up
S: Move Down
A: Move Left
D: Move Right
q: Quit
```
